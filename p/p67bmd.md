# Problem 67b

Consider a string representation of a tree with a single character naming each node such as:

```a(b(d,e),c(,f(g,)))```

Write a function to generate a tree from its string representation created in [Problem 67a](p67a.md). 

## Example
```elm
toTree "x(y,a(,b))" ==  
    Node 'x' 
        (Node 'y' Empty Empty) 
        (Node 'a' 
            Empty 
            (Node 'b' Empty Empty))
```
