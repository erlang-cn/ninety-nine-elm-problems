# Problem 3 Solutions

## Solution 1

```
{-| return the nth item of a list (one-relative index)
-}
elementAt : List a -> Int -> Maybe a
elementAt list n =
    if n < 1 then
        Nothing
    else
        case List.drop (n - 1) list of
            [] ->
                Nothing

            y :: ys ->
                Just y
```
